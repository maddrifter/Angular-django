Extras

GitHub

Sync your Github Issues with ScrumDo cards. Reference ScrumDo cards in your commit messages.



Tip: If you are using Github with multiple workspace, you can configure Github for your entire organization.

Slack

Send status updates to a Slack channel.

HipChat

Send status updates to a HipChat room.

FlowDock

Send status updates to your FlowDock Team Inbox.

Harvest

Track your time with Harvest.

Email a Card

Send an email to a special address to create a new card

Others

The old Jira and Basecamp extras are not available. If you require this functionality, please consider using Classic ScrumDo.

Our apologies for any trouble this may cause.
