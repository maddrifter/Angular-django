Organization Planning Setup

Release Planning

Create releases, assign cards to releases across workspaces, and track progress of those releases.

This is the simpler option that requires minimal overhead and is most useful for organizations who want to do bottom-up release planning.

Changing from Value Stream to Release planning mode

When you go from Value Stream to Release planning mode, the organization wide board will go away, and any cards on that board will be moved into the value stream backlog. They will then start appearing as Releases in release mode.

NOTE: This means the position of those milestones on your organization wide board, and what release they were part of will be lost if you later decide to switch back to value stream mode.

Read more here: http://help.scrumdo.com/help/releases.html

Only staff users can enable releases.

Value Stream Planning beta

Create high level milestones, and track those milestones across an organization level board. Assign milestones to workspace teams to work on. Group milestones into releases and track progress of those milestones and releases.

This is most useful for an organization who wants to do top-down planning. It can give more control and visibility than the Release Planning option.

Switching from Release to Value Stream planning mode

This is completely safe. Your current releases will appear in the backlog of your organization wide board as milestones, and any cards assigned to them will continue to be assigned to them. You can then move those milestones to your organization board and use all the value stream planning tools with them.

Read more here: http://help.scrumdo.com/help/valuestream.html

Only staff users can enable value stream planning.
